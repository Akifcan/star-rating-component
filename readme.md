# Javascript Star Rating Component

## Usage

```
    <star-rating
        color="red"
        stroke size="14"
        stars="3.5"
        maxStar="5"></star-rating>
    <star-rating
        size="16"
        stars="3.5"
        maxStar="5"></star-rating>
    <star-rating
        color="blue"
        size="16"
        stars="2.5"
        maxStar="5"></star-rating>
```
